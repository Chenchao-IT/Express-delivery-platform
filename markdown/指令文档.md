# 校园快递配送系统 - 项目启动指令文档

## 一、环境要求

| 组件 | 版本要求 |
|------|----------|
| JDK | **17**（必须 JDK，Lombok 与 Java 25 不兼容） |
| MySQL | 8.0 |
| Node.js | 18+ |
| Redis | 可选，用于登录限流（无 Redis 时使用 no-redis 配置） |

---

## 二、数据库准备

### 1. 创建数据库

```sql
CREATE DATABASE IF NOT EXISTS campus_express DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE campus_express;
```

### 2. 执行建表脚本

```bash
mysql -u root -p < database/schema.sql
```

或在 Navicat / MySQL 客户端中执行 `database/schema.sql`。

### 3. 修改数据库密码（可选）

编辑 `backend/src/main/resources/application.yml`：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/campus_express?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true
    username: root
    password: 你的MySQL密码
```

---

## 三、启动后端

> 后端默认端口：**8080**，context-path：`/api`

### 方式一：使用 bat 脚本（推荐，自动设置 JDK 17）

```cmd
cd D:\毕业论文\code\backend
set-java-and-run.bat
```

或直接**双击** `backend/set-java-and-run.bat`。

### 方式二：使用 PowerShell 脚本

```powershell
cd D:\毕业论文\code\backend
.\run.ps1
```

### 方式三：命令行直接启动（需已配置 JAVA_HOME 或 Maven）

```powershell
cd D:\毕业论文\code\backend
mvn clean spring-boot:run
```

### 方式四：无 Redis 模式（未安装 Redis 时）

```powershell
cd D:\毕业论文\code\backend
mvn spring-boot:run -Dspring-boot.run.arguments="--spring.profiles.active=no-redis"
```

### 方式五：指定 JDK 17 后启动

```powershell
$env:JAVA_HOME = "C:\Program Files\Eclipse Adoptium\jdk-17.0.13.11-hotspot"  # 按实际安装路径修改
cd D:\毕业论文\code\backend
mvn clean spring-boot:run
```

**启动成功标志**：终端输出 `Tomcat started on port 8080`、`Started ExpressDeliveryApplication`。

## 六、常用指令速查

### 清空登录限流（出现「登录过于频繁」时）

**方式一**：登录页点击「限流了？点击清空」按钮。

**方式二**：浏览器控制台执行（需通过前端代理）：
```javascript
fetch('/api/auth/rate-limit/clear', { method: 'POST' }).then(r => r.json()).then(console.log)
```

**方式三**：curl 直接调用后端：
```bash
curl -X POST http://localhost:8080/api/auth/rate-limit/clear
```

### 重置数据库测试数据

```sql
USE campus_express;
SET FOREIGN_KEY_CHECKS = 0;
TRUNCATE delivery_task;
TRUNCATE package;
DELETE FROM user WHERE username != 'admin';
SET FOREIGN_KEY_CHECKS = 1;
```

重启后端后，DataInitializer 会自动重新创建学生、快递员、包裹、配送任务等测试数据。

### 手动插入测试用户（可选）

```bash
mysql -u root -p campus_express < database/seed-data.sql
```

> 注意：seed-data.sql 中的密码哈希可能不正确，建议依赖 DataInitializer 自动创建。

## 七、环境变量（可选）

| 变量 | 说明 |
|------|------|
| MYSQL_PASSWORD | MySQL 密码，覆盖 application.yml |
| JAVA_HOME | JDK 安装路径（如 `C:\Program Files\Eclipse Adoptium\jdk-17.0.13.11-hotspot`） |
| REDIS_HOST | Redis 主机（空则可能使用内存限流） |

---

## 八、一键启动流程

1. 启动 MySQL，确保 `campus_express` 数据库存在  
2. 启动 Redis（若使用）或使用 `--spring.profiles.active=no-redis`  
3. 启动后端：`cd backend` → `set-java-and-run.bat` 或 `mvn spring-boot:run`  
4. 启动前端：`cd frontend` → `npm run dev`  
5. 浏览器访问 http://localhost:5173
